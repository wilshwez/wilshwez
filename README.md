## 🧠 Featured Projects

---

### 🔍 Log-Based Intrusion Detection & Forensics Tool (LIDFT)
A security-focused system built for SOC workflows:
- Parses and analyzes system logs in real time
- Detects anomalies using rule-based and intelligent techniques
- Provides a Flask-based monitoring dashboard
- Generates real-time alerts for suspicious activity
- Supports admin-level log investigations and forensic analysis

> Built with security monitoring and SOC operations in mind.

---

### 🛡️ Advanced Adware & Browser Hijacker Scanner
An open-source Python-based security scanner for Windows, macOS, and Linux:
- Scans running processes, registry, browser history, extensions, and network connections
- Matches against 54+ verified threat signatures sourced from Malwarebytes, Trend Micro, and BleepingComputer
- Auto-remediates threats — terminates malicious processes and quarantines files with AES encryption
- Generates colour-coded terminal output and HTML reports
- Real-time process monitor detects new threats as they launch

> No antivirus subscription required. Fully open source.

---

### 🔌 API Security Analyzer & Endpoint Discovery Tool *(In Development)*
A security tool designed to identify and analyze web API attack surfaces:
- Discovers hidden REST, SOAP, and GraphQL endpoints through intelligent fuzzing
- Analyzes API responses for vulnerabilities including IDOR, broken authentication, and injection
- Supports parameter fuzzing for query, path, and request body parameters
- GraphQL introspection support to automatically map full API schemas
- WSDL parsing for SOAP endpoint and parameter discovery
- Generates structured vulnerability reports

> Built for penetration testers and SOC analysts performing API security assessments.
